# 📁 <b><a style="color:#00adb5" href="https://swexpertacademy.com/main/code/problem/problemDetail.do?contestProbId=AV5PobmqAPoDFAUq" target=_blank>[D2_1954] 달팽이 숫자</a></b>

```java
import java.util.Scanner;
 
public class Solution {
    public static void main(String[] args) throws Exception{
        Scanner sc = new Scanner(System.in);
        int T = sc.nextInt();
 
        for (int tc = 1; tc <= T; tc++) {
 
            // 입력
            int N = sc.nextInt();

            // 수를 저장할 배열
            int[][] arr = new int[N][N];

            // 수 
            int num = 1;
             

            // 입력이 1인 경우 -> '1' 출력
            if(N == 1) {
                arr[N-1][N-1] = 1;
            }
 
            // N이 짝수인 경우
            if (N / 2 == 0) {
                for (int j = 0; j < N / 2; j++) {

                    // 위에 가로
                    for (int i = j; i < N - (j + 1); i++) {
                        arr[j][i] = num;
                        num++;
                    }
 
                    // 오른쪽 세로
                    for (int i = j; i < N - (j + 1); i++) {
                        arr[i][N - (j + 1)] = num;
                        num++;
                    }
 
                    // 밑에 가로
                    for (int i = N - (j + 1); i > j; i--) {
                        arr[N - (j + 1)][i] = num;
                        num++;
                    }
 
                    // 왼쪽 세로
                    for (int i = N - (j + 1); i > j; i--) {
                        arr[i][j] = num;
                        num++;
                    }
                }
 
            }
            // N이 홀수인 경우
            else {
                for (int j = 0; j < N / 2; j++) {
 
                    // 위에 가로
                    for (int i = j; i < N - (j + 1); i++) {
                        arr[j][i] = num;
                        num++;
                    }
 
                    // 오른쪽 세로
                    for (int i = j; i < N - (j + 1); i++) {
                        arr[i][N - (j + 1)] = num;
                        num++;
                    }
 
                    // 밑에 가로
                    for (int i = N - (j + 1); i > j; i--) {
                        arr[N - (j + 1)][i] = num;
                        num++;
                    }
 
                    // 왼쪽 세로
                    for (int i = N - (j + 1); i > j; i--) {
                        arr[i][j] = num;
                        num++;
                    }

                    // 마지막 중앙값
                    if (arr[N / 2][N / 2] == 0) {
                        arr[N / 2][N / 2] = N * N;
                    }
                }
            }

            // 출력
            System.out.println("#"+ tc);
            for (int i = 0; i < N; i++) {
                for (int j = 0; j < N; j++) {
                    System.out.print(arr[i][j] + " ");
                }
                System.out.println();
            }
        }
 
    }
 
}

```

## 🤔 <b><a style="color:#00adb5">나의 생각</a></b>
사각형을 한 변씩 나누어 계산해 주었다. <br>
N이 짝수일 경우, 홀수일 경우 나누어서 반복문을 사용하였는데,<br>
뭔가 재귀 함수를 통해서도 할 수 있을 것 같다.<br>
그리고 N이 1인 경우를 처음에 놓치고 틀린게 좀 아쉽다.
